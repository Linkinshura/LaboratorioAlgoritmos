# Tips de Codificacion extras:

## Acumulador:
acum = acum + var;

es lo mismo que

acum+= var;

## Iteracion:
cant= cant + 1;

es lo mismo que

cant++;

Si, usamos esto mismo en el for con el i++

## Espaciado para mejorar comprension

Usamos el printf("\n");

para que en la consola se muestre un espacio entre los printf

tambien el uso de separar por espacios
